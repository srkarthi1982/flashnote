# Contributing to FlashNote

Thank you for your interest in improving FlashNote.

1. **Fork** this repository.
2. Create a new branch:
   ```bash
   git checkout -b feature/my-change
   ```
3. Make your changes.
4. Run the app locally:
   ```bash
   npm install
   npm run dev
   ```
5. Verify the build:
   ```bash
   npm run build
   ```
6. Commit and push:
   ```bash
   git commit -m "Describe your change"
   git push origin feature/my-change
   ```
7. Open a **Pull Request** and clearly describe:
   - What you changed
   - Why you changed it
   - Any impact on data or users
